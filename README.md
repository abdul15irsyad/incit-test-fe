# INCIT Test Frontend

## Tech Stack

- Next JS 14
- Tailwind CSS

## Installation

1. install dependencies

   ```bash
   npm install
   ```

2. setup `.env` file and set the `NEXT_PUBLIC_API_URL`

   ```bash
   cp .env.example .env
   ```

3. run the application

   ```bash
   npm run dev

   # or

   npm run build && npm run start
   ```
